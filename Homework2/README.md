本次作业满分15分，截止日期为2024.4.18，需要实现任意一种网格处理算法，可以从第3~6课PPT中的算法任选，包括网格平滑、网格简化、参数化、网格变形等，也可以选择其他相关的算法进行实现。

如拉普拉斯平滑、顶点/法向双边滤波、稀疏优化等

如泊松变形、拉普拉斯变形、ARAP变形等
